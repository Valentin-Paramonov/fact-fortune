Fact Fortune
============

### Store [FactSlides][1] facts as a [fortune][2] file

#### Install
```
npm install
```
#### Start [RethinkDB][3] server
```
rethinkdb --daemon
```
#### Run update script
```
./update-fortune.sh
```

[1]: http://factslides.com
[2]: https://en.wikipedia.org/wiki/Fortune_(Unix)
[3]: http://www.rethinkdb.com
